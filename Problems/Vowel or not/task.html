Vowel or not Open on JetBrains Academy

Write a function that checks whether an English letter is a vowel. The input letter can be in any case.

Do not consider the letter 'y' to be a vowel.

Your function is to take a value of the Char type and return a Boolean.

 

Use the provided code template.

Examples

1) isVowel('a') should be true

2) isVowel('A') should be true

3) isVowel('b') should be false
